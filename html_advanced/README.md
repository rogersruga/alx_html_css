# HTML & CSS — Homepage Styling Project

This project is a focused exercise in CSS fundamentals and layout: building and styling a homepage for a class assignment while practicing modern CSS techniques and best practices.

What you'll find in this workspace
- A simple homepage and base template to style while learning layout, typography, colors, and responsive design.
- Example CSS files and an extra sample layout for practice.

Quick overview
- Purpose: Style a homepage for a class project to solidify core CSS skills (selectors, box model, flex/grid, responsive breakpoints).
- Primary files to edit and review:
  - [html_advanced/index.html] — the homepage to style.
  - [html_advanced/base.html] — base template used as a reference.
  - [html_advanced/README.md] — this file.
  - [html_advanced/images] — project image assets.

How to preview locally
1. Open [html_advanced/index.html] in your browser (or use your editor's Live Preview).
2. Modify CSS in [css_basic/styles.css] or create a dedicated stylesheet and link it in [html_advanced/index.html].
3. Refresh the page to see changes.

Learning goals
- Use selectors effectively and keep styles maintainable.
- Practice layout with Flexbox and Grid.
- Apply responsive rules to ensure mobile-friendly design.
- Improve visual hierarchy with typography, spacing, and color.
